# 🌾 Crop Recommendation and Yield Prediction System

## 📌 Overview

The **Crop Recommendation and Yield Prediction System** is an AI-powered machine learning project that helps farmers make informed decisions by recommending the most suitable crop and predicting crop yield based on environmental and soil parameters.

The system uses machine learning algorithms to analyze features such as nitrogen, phosphorus, potassium levels, temperature, humidity, pH, and rainfall to provide accurate recommendations and yield predictions.

---

## 🚀 Features

* Crop recommendation based on soil and climate conditions
* Crop yield prediction using machine learning
* Data preprocessing and feature engineering
* Exploratory Data Analysis (EDA)
* Correlation heatmap visualization

* User-friendly prediction system
